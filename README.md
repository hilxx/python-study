# 파이썬 기초 학습 레포지토리

파이썬 프로그래밍 기초를 학습하기 위한 연습 레포지토리입니다.

## 파이썬이란?
- 인터프리터 언어: 한 줄씩 실행
- 동적 타입: 변수 타입을 미리 선언하지 않음
- 웹, 데이터 분석, 자동화, AI 등 거의 모든 분야에서 사용
👉 현재 기준 권장 버전: Python 3.10 이상 (260204)

## 📚 학습 목표

- 파이썬 기본 문법 이해
- 프로그래밍 기본 개념 습득
- 실습 문제를 통한 실전 연습
- 문제 해결 능력 향상

## 🎯 학습 내용

### 1. 기본 문법
- 변수와 데이터 타입
- 연산자
- 입력과 출력
- 주석 작성

### 2. 제어문
- 조건문 (if, elif, else)
- 반복문 (for, while)
- break와 continue

### 3. 자료구조
- 리스트 (List)
- 튜플 (Tuple)
- 딕셔너리 (Dictionary)
- 집합 (Set)

### 4. 함수
- 함수 정의와 호출
- 매개변수와 반환값
- 지역 변수와 전역 변수

### 5. 객체지향 프로그래밍 기초
- 클래스와 객체
- 메서드
- 생성자

## 📁 파일 구조

```
python-styudy/
├── README.md          # 프로젝트 설명서
├── main.py            # 메인 실행 파일
└── (추가 예정)        # 연습 문제 및 예제 파일들
```

## 🚀 사용 방법

1. 각 주제별 예제 파일을 열어 코드를 확인합니다.
2. 코드를 실행해보고 결과를 확인합니다.
3. 주석을 읽으며 각 코드의 동작을 이해합니다.
4. 연습 문제를 직접 풀어봅니다.

### 코드 실행 방법

터미널에서 다음 명령어로 실행할 수 있습니다:

```bash
python main.py
```

## 📝 학습 순서 제안

1. **기본 문법**부터 시작하여 차근차근 학습합니다.
2. 각 주제를 학습한 후 **연습 문제**를 풀어봅니다.
3. 모르는 부분은 **주석**을 참고하거나 검색해봅니다.
4. 직접 코드를 **수정하고 실행**해보며 실험합니다.

## 💡 학습 팁

- 코드를 그대로 따라 치는 것보다 **이해하며 작성**하는 것이 중요합니다.
- 에러가 발생하면 **에러 메시지를 읽고** 해결 방법을 찾아봅니다.
- 작은 프로그램이라도 **직접 만들어보는 것**이 실력 향상에 도움이 됩니다.
- 모르는 개념이 나오면 **공식 문서**나 **튜토리얼**을 참고합니다.
